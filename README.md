# **Rocket Race**

> WebGL Unity Game. Collect stars and don't crash into the the planets!

<br />

### [Play](https://sunghogo.github.io/Rocket-Race/)

### Controls:

| Key | Controls |
| --- | -------- |
| W or ↑ | Rotate Up |
| S or ↓ | Rotate Down |
| A or ← | Rotate Left |
| D or → | Rotate Right |
| SPACE | Thrust |
| ESC | Quit (Only for .exe) |
